Welcome to SmartCart

Your streamlined solution for managing product inventory with the power of the MEAN stack.

🛍️ What Is SmartCart?

SmartCart is a full-stack web application built to simplify product management for developers and small businesses. It offers a clean interface and robust backend to handle everything from adding new items to updating and removing them.

✨ Key Features

  • Create and manage product listings
  • Edit product details in real time
  • Delete items with a single click
  • View all products in a responsive, sortable table

🔧 Built With

  • MongoDB – Flexible NoSQL database for storing product data
  • Express.js – Lightweight backend framework for routing and APIs
  • Angular – Dynamic frontend for seamless user interaction
  • Node.js – Scalable runtime environment for server-side logic

🚀 Why It Matters

SmartCart is more than a demo—it's a launchpad for building scalable e-commerce platforms. Whether you're learning full-stack development or prototyping a real-world solution, this app gives you the tools to succeed.
